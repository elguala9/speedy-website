How it works
01

Index once, search forever

Speedy chunks and embeds your entire codebase into a local SQLite vector store powered by Ollama. Fully offline, no API keys required.

02

Always up to date

A single lightweight daemon watches all your workspaces and updates the index the moment files change — no manual re-indexing needed.

03

Plug into any AI agent

An MCP server lets Claude Code, Cursor, Windsurf, and other agents query your code semantically — getting smarter, more relevant context with every request.

04

Desktop GUI included

Manage workspaces, monitor the daemon, and inspect live logs from a clean native interface. No terminal required to get started.
